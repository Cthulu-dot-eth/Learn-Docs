# DNS Imports

To import a DNS domain name into ENS does not cost any yearly registration fees. However, importing DNS domains does cost a fair amount of gas currently. Imported DNS domain names function the same way as ENS names, having their record setup, including payment addresses, avatar records, social media links, and more.
